# 簡易購物網站

使用 React+.Net5 開發

## 使用技術

### 前端

React

- Material UI
- Axios

### 後端

.Net5 Webapi

- Swagger
- Identity + Jwt Token + Refresh Token
- EF Core(Code First)
- Repository + UnitOfWork
- Email Server
- SpaServices
- Image Server

### 網站架設

Amazon Web Services

- EC2 Linux 2
- RDS PostgreSQL
- S3
